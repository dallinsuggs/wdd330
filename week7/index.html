Week 7 notes

  • IIFE functions are really useful for controlling scope.
  • If you use strict mode, put all your code inside an IIFE function to avoid applying strict mode to anyone else's code you end up using.
  • Recursive functions are powerful, but be careful (make sure the loop ends at some point!)
  • AJAX fetch requests can receive responses in various formats, including HTML and json.
  • To customize requests, we can create and provide a request object with its own properties.
  • Any real reason to use a patch request over a put request? Just if the data item is really big? For small stuff it seems like patch wouldn't be worth the trouble.
  • AJAX requests can not only request information to display on a web page, but also send data to update a database.
