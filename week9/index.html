Week 9 notes

  • Global object in a browser is the window object.
  • Global variables can be initiated without let or var, and then are access like "varName" or "window.varName".
  • Don't use dialogs unless absolutely necessary - they are aggravating for the user, and aren't reliable.
  • window.history.go(int) allows you to access browser history by navigating a specific number of pages forward or backward.
  • Why is document.write() frowned upon?
  • The data- attribute allows us to store custom private data in the page.
  • localStorage is part of Web Storage API. It allows storing data, like cookies, but there isn't a data limit and the data isn't shared with the server for each request.
  • The navigator object of Geolocation API can be used to get geographical location with the getCurrentPosition() method.
  • They probably use navigator or something similar for Google Maps? The position object of the getCurrentPosition() method has properties like speed, altitude, heading, etc.
  • workers allow for multitasking or concurrency.
  • The websocket protocol allows for push messaging - useful for chat programs, as the server can "push" replies to the client.
  • Media API allows for management of multimedia content in a webpage.
  • How was the battery API used for user profiling?
