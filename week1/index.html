Week 1 notes

  • Start small, then scale up.
  • Reduce required steps to help users acheive their goals.
  • Design with slow internet in mind.
  • How many images is too many for loading times?
  • Reduce user cognitive load.
  • "Is there a more streamlined way to provide this option?"
  • Use empathy to see things from the user's perspective.
